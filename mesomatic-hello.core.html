mesomatic-hello.core

The namespace which holds the entry point for the hello-world demo.

-main

(-main master task-type)

It is expected that this function be called from lein in the following manner:

$ lein mesomatic 127.0.0.1:5050 <task type>

where <task-type> is one of:

  • executor
  • framework

That being said, only Mesos should call with the executor task type; calling humans will only call with the framework task type.

Note that in order for this to work, one needs to add the following alias to the project’s project.clj:

:aliases {"mesomatic" ["run" "-m" "mesomatic-hello.core"]}
